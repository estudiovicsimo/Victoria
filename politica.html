Política de Privacidade

Última atualização: Março de 2026

Esta Política de Privacidade descreve como as informações dos visitantes são coletadas, utilizadas e protegidas ao acessar este site.

Ao navegar por este site ou entrar em contato por meio do formulário disponível, você concorda com as práticas descritas nesta política.

1. Coleta de informações

Este site pode coletar algumas informações fornecidas diretamente pelo usuário, como:

  • Nome
  • E-mail
  • Mensagem enviada através do formulário de contato

Esses dados são fornecidos voluntariamente pelo visitante quando ele decide entrar em contato para solicitar informações, orçamento ou iniciar uma conversa profissional.

Além disso, algumas informações técnicas podem ser coletadas automaticamente durante a navegação, como:

  • Endereço IP
  • Tipo de navegador
  • Dispositivo Utilizado
  • Páginas visitadas no site

Esses dados são utilizados apenas para fins de análise de navegação e melhoria da experiência do usuário.

2. Uso das informações

As informações coletadas podem ser utilizadas para:

  • Responder mensagens enviadas pelo formulário de contato
  • Entrar em contato para responder dúvidas ou enviar propostas de trabalho
  • Melhorar a experiência de navegação no site
  • Analisar o desempenho e uso das páginas

Os dados não são vendidos, alugados ou compartilhados com terceiros para fins comerciais.

3. Compartilhamento de dados

As informações fornecidas pelos usuários não são compartilhadas com terceiros, exceto quando necessário para:

  • Cumprimento de obrigações legais
  • Proteção de direitos legais do responsável pelo site

4. Links para sites externos

Este site pode conter links para plataformas externas, como redes sociais ou portfólios online.

Ao acessar esses links, o usuário será direcionado para ambientes que possuem suas próprias políticas de privacidade. Este site não se responsabiliza pelas práticas de privacidade desses serviços.

5. Segurança das informações

Medidas razoáveis são adotadas para proteger as informações fornecidas pelos usuários contra acesso não autorizado, alteração, divulgação ou destruição.

No entanto, nenhum sistema de transmissão ou armazenamento de dados na internet é totalmente seguro.

6. Direitos do usuário

De acordo com a Lei Geral de Proteção de Dados (LGPD), o usuário pode solicitar a qualquer momento:

  • Acesso aos seus dados
  • Correção de informações
  • Exclusão dos dados fornecidos

Para isso, basta entrar em contato pelos canais informados neste site.

7. Alterações nesta política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças no funcionamento do site ou em requisitos legais. Recomenda-se a revisão desta página ocasionalmente.

8. Contato

Em caso de dúvidas sobre esta Política de Privacidade ou sobre o uso de dados neste site, entre em contato por meio do formulário disponível na página de contato.
